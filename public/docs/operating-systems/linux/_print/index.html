Pre-requirements

  • Any linux distribution (ubuntu/redhat/or any)
  • VM/Metal/Containers
  • Spin up instance in any cloud (openstack/ or any)
  • WSL windows subsystem for linux
Don’t try installing software if you are running the live CD. Because writeable space on a live CD uses virtual memory (RAM), that space is limited and can easily run out. Also, when you reboot your system, anything you install will be gone.

yum config-manager –add-repo=/mnt/BaseOS yum config-manager –add-repo=/mnt/AppStream yum config-manager –add-repo=http://127.0.0.1/baseOS yum config-manager –add-repo=http://127.0.0.1/AppStream

Note: this will create a repo file under /etc/yum.repo.d/ yum repolist
